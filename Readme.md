Instructions on How to Run the Program


1. Make sure you have MySQL as well as XAMPP installed.

2. Import the sql files in database folder. Start with greenloop.sql

3. Rename src/main/resources/config.properties.example as config.properties.

4. Add a gmail and a app password to the config.properties file.

5. Run src/main/java/com.greenloop/Main.java to open the program.
